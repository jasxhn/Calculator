# 🧮 Simple Calculator

A responsive calculator built using **HTML, CSS, and JavaScript**. It supports basic arithmetic operations with a clean user interface and mobile-friendly design.

## 🚀 Features

* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)
* Decimal number support
* Clear display functionality
* Responsive design
* Simple and lightweight

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## ⚙️ How to Run

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Navigate to the project folder:

```bash
cd calculator
```

3. Open `index.html` in your browser.
